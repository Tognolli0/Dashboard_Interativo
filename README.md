# Dashboard Interativo

Dashboard corporativo em Blazor Server com foco em indicadores operacionais, cards de desempenho, graficos e exportacao de relatorios.

## Stack

- ASP.NET Core Blazor Server
- .NET 10
- MudBlazor
- Chart.js
- QuestPDF

## Estrutura

- `DashboardMamix`: aplicacao principal
- `Dashboard.sln`: solucao do projeto

## Status

Projeto com dados mockados e arquitetura pronta para integrar com fonte de dados real no futuro.
O arquivo `DashboardMamix/appsettings.Local.example.json` serve como ponto de partida para configuracao local.

## Objetivo de portfolio

Este repositorio funciona bem como demonstracao de dashboard corporativo e camada de apresentacao em Blazor.
